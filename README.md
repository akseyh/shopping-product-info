# shopping-product-info

## Installing
```bash
$ npm install shopping-product-info
```

## Usage
```
const getProductInfo = require('shopping-product-info');
```

## Example
It's working with trendyol.com, hepsiburada.com yet.

```
async function ...() {
  let product = getProductInfo(url);
  console.log(product);
}
```
